# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅ Yes |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: [darkdevworker@users.noreply.github.com](mailto:darkdevworker@users.noreply.github.com)
3. Include steps to reproduce
4. Allow reasonable time for a fix before public disclosure

## Security Measures

- API keys are stored locally with file permissions (chmod 600)
- No telemetry or external data collection
- All downloads go through HTTPS
- SHA256 hash verification available for all downloads
